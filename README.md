# 🎧 Spotify Integration

Website ini menggunakan **Spotify Embed** untuk menampilkan album langsung dari Spotify.

Contoh implementasi:

```
<iframe 
src="https://open.spotify.com/embed/album/ALBUM_ID"
width="300"
height="380"
frameborder="0"
allow="encrypted-media">
</iframe>
```

Dengan fitur ini pengunjung dapat:

* melihat album artis
* memutar preview lagu
* membuka album langsung di Spotify
